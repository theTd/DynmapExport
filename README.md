# DynmapExport
concat tiles rendered by dynmap to a large image.

  Usage:
  
|Option|Description|
|------|:-----------:|
|--db-password <String>|
|--db-url <String>|eg jdbc:mysql://localhost/mc|
|--db-user <String>||
|--map-id <String>|eg "flat"|
|--table-prefix <String>|eg "dynmap_build_"|
|--threads <Integer>|working threads, defaults to 10|
|--world <String>||
|--zoom <Integer>||
